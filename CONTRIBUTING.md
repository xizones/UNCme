# How to add code/contribute
1. Join this discord server: https://discord.gg/RjNNNt2hg2
2. go to #addition-requests
3. drop in your code snippet (preferably as a .lua or .luau file)

**You will be credited for you work!**

## What should my test scipts include
Your test script should return a table with the following information:
* The name of the function (string)
* wether or not the function passed (boolean)
* an optional comment (string)

There are examples of good test scripts in this repository that you can reference
